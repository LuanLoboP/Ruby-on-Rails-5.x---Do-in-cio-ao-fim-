- Instalar 
    gem install pry
    pry = no terminal para iniciar a ferramenta

- 2 Beneficios do REPL pry:
    Identação no terminal
    Highlighting = Coloração no terminal para o código

- Se quero usar ele como padrão ao chamar irb:
    Criar um arquivo .irbcrc na home do usuário
        touch ~/.irbrc
    Dentro do .irbc 
        require 'rubygems'
        require 'pry'
        Pry.start
        exit
        (:qw = quit e write)
